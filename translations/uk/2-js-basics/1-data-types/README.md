### –ê—Ä–∏—Ñ–º–µ—Ç–∏—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏

–ê—Ä–∏—Ñ–º–µ—Ç–∏—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –¥–æ–∑–≤–æ–ª—è—é—Ç—å –≤–∏–∫–æ–Ω—É–≤–∞—Ç–∏ –º–∞—Ç–µ–º–∞—Ç–∏—á–Ω—ñ –æ–±—á–∏—Å–ª–µ–Ω–Ω—è –≤ JavaScript. –¶—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –¥–æ—Ç—Ä–∏–º—É—é—Ç—å—Å—è —Ç–∏—Ö —Å–∞–º–∏—Ö –ø—Ä–∏–Ω—Ü–∏–ø—ñ–≤, —è–∫—ñ –º–∞—Ç–µ–º–∞—Ç–∏–∫–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞–ª–∏ —Å—Ç–æ–ª—ñ—Ç—Ç—è–º–∏ ‚Äì —Ç—ñ —Å–∞–º—ñ —Å–∏–º–≤–æ–ª–∏, —è–∫—ñ –∑'—è–≤–ª—è–ª–∏—Å—è –≤ —Ä–æ–±–æ—Ç–∞—Ö —Ç–∞–∫–∏—Ö —É—á–µ–Ω–∏—Ö, —è–∫ –ê–ª—å-–•–æ—Ä–µ–∑–º—ñ, —è–∫–∏–π —Ä–æ–∑—Ä–æ–±–∏–≤ –∞–ª–≥–µ–±—Ä–∞—ó—á–Ω—É –Ω–æ—Ç–∞—Ü—ñ—é.

–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∞—Ü—é—é—Ç—å —Ç–∞–∫, —è–∫ –≤–∏ –æ—á—ñ–∫—É—î—Ç–µ –≤—ñ–¥ —Ç—Ä–∞–¥–∏—Ü—ñ–π–Ω–æ—ó –º–∞—Ç–µ–º–∞—Ç–∏–∫–∏: –ø–ª—é—Å –¥–ª—è –¥–æ–¥–∞–≤–∞–Ω–Ω—è, –º—ñ–Ω—É—Å –¥–ª—è –≤—ñ–¥–Ω—ñ–º–∞–Ω–Ω—è —Ç–æ—â–æ.

–Ü—Å–Ω—É—î –∫—ñ–ª—å–∫–∞ —Ç–∏–ø—ñ–≤ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ñ–≤ –¥–ª—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–Ω–∏—Ö —Ñ—É–Ω–∫—Ü—ñ–π, —ñ –¥–µ—è–∫—ñ –∑ –Ω–∏—Ö –Ω–∞–≤–µ–¥–µ–Ω—ñ —Ç—É—Ç:

### –†—è–¥–∫–∏

–£ JavaScript —Ç–µ–∫—Å—Ç–æ–≤—ñ –¥–∞–Ω—ñ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ñ —è–∫ —Ä—è–¥–∫–∏. –¢–µ—Ä–º—ñ–Ω "—Ä—è–¥–æ–∫" –ø–æ—Ö–æ–¥–∏—Ç—å –≤—ñ–¥ –∫–æ–Ω—Ü–µ–ø—Ü—ñ—ó —Å–∏–º–≤–æ–ª—ñ–≤, –∑'—î–¥–Ω–∞–Ω–∏—Ö —É –ø–æ—Å–ª—ñ–¥–æ–≤–Ω—ñ—Å—Ç—å, —è–∫ —Å–µ—Ä–µ–¥–Ω—å–æ–≤—ñ—á–Ω—ñ –ø–∏—Å–∞—Ä—ñ –∑'—î–¥–Ω—É–≤–∞–ª–∏ –ª—ñ—Ç–µ—Ä–∏, —â–æ–± —É—Ç–≤–æ—Ä–∏—Ç–∏ —Å–ª–æ–≤–∞ —Ç–∞ —Ä–µ—á–µ–Ω–Ω—è —É —Å–≤–æ—ó—Ö —Ä—É–∫–æ–ø–∏—Å–∞—Ö.

**–†–æ–∑—É–º—ñ–Ω–Ω—è —Ü–∏—Ö –∫–æ–Ω—Ü–µ–ø—Ü—ñ–π:**
- **–í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î** –æ–¥–∏–Ω–∞—Ä–Ω—ñ –ª–∞–ø–∫–∏ `'` –∞–±–æ –ø–æ–¥–≤—ñ–π–Ω—ñ –ª–∞–ø–∫–∏ `"` –¥–ª—è –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è —Ä—è–¥–∫—ñ–≤

- **–ü—Ä–∏—Å–≤–æ—é—î** —Ä—è–¥–∫–æ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –∑–º—ñ–Ω–Ω–∏–º –¥–ª—è –ø–æ–¥–∞–ª—å—à–æ–≥–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è
- **–í–∏–º–∞–≥–∞—î** –ª–∞–ø–æ–∫, —â–æ–± –≤—ñ–¥—Ä—ñ–∑–Ω–∏—Ç–∏ —Ç–µ–∫—Å—Ç –≤—ñ–¥ –Ω–∞–∑–≤ –∑–º—ñ–Ω–Ω–∏—Ö

–ù–µ –∑–∞–±—É–¥—å—Ç–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –ª–∞–ø–∫–∏ –ø—Ä–∏ –Ω–∞–ø–∏—Å–∞–Ω–Ω—ñ —Ä—è–¥–∫–∞, —ñ–Ω–∞–∫—à–µ JavaScript –ø—Ä–∏–ø—É—Å—Ç–∏—Ç—å, —â–æ —Ü–µ –Ω–∞–∑–≤–∞ –∑–º—ñ–Ω–Ω–æ—ó.

### –§–æ—Ä–º–∞—Ç—É–≤–∞–Ω–Ω—è —Ä—è–¥–∫—ñ–≤

–ú–∞–Ω—ñ–ø—É–ª—è—Ü—ñ—è —Ä—è–¥–∫–∞–º–∏ –¥–æ–∑–≤–æ–ª—è—î –æ–±'—î–¥–Ω—É–≤–∞—Ç–∏ —Ç–µ–∫—Å—Ç–æ–≤—ñ –µ–ª–µ–º–µ–Ω—Ç–∏, –≤–∫–ª—é—á–∞—Ç–∏ –∑–º—ñ–Ω–Ω—ñ —Ç–∞ —Å—Ç–≤–æ—Ä—é–≤–∞—Ç–∏ –¥–∏–Ω–∞–º—ñ—á–Ω–∏–π –∫–æ–Ω—Ç–µ–Ω—Ç, —è–∫–∏–π —Ä–µ–∞–≥—É—î –Ω–∞ —Å—Ç–∞–Ω –ø—Ä–æ–≥—Ä–∞–º–∏. –¶—è —Ç–µ—Ö–Ω—ñ–∫–∞ –¥–æ–∑–≤–æ–ª—è—î –ø—Ä–æ–≥—Ä–∞–º–Ω–æ —Å—Ç–≤–æ—Ä—é–≤–∞—Ç–∏ —Ç–µ–∫—Å—Ç.

–ß–∞—Å—Ç–æ –≤–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –æ–±'—î–¥–Ω–∞—Ç–∏ –∫—ñ–ª—å–∫–∞ —Ä—è–¥–∫—ñ–≤ ‚Äì —Ü–µ–π –ø—Ä–æ—Ü–µ—Å –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è –∫–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü—ñ—î—é.

–©–æ–± **–∫–æ–Ω–∫–∞—Ç–µ–Ω—É–≤–∞—Ç–∏** –¥–≤–∞ –∞–±–æ –±—ñ–ª—å—à–µ —Ä—è–¥–∫—ñ–≤, –∞–±–æ –æ–±'—î–¥–Ω–∞—Ç–∏ —ó—Ö, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä `+`.

```javascript
let myString1 = "Hello";
```

**–ö—Ä–æ–∫ –∑–∞ –∫—Ä–æ–∫–æ–º, –æ—Å—å —â–æ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è:**
- **–û–±'—î–¥–Ω—É—î** –∫—ñ–ª—å–∫–∞ —Ä—è–¥–∫—ñ–≤ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `+`
- **–ó'—î–¥–Ω—É—î** —Ä—è–¥–∫–∏ –±–µ–∑ –ø—Ä–æ–±—ñ–ª—ñ–≤ —É –ø–µ—Ä—à–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ
- **–î–æ–¥–∞—î** –ø—Ä–æ–±—ñ–ª–∏ `" "` –º—ñ–∂ —Ä—è–¥–∫–∞–º–∏ –¥–ª—è —á–∏—Ç–∞–±–µ–ª—å–Ω–æ—Å—Ç—ñ
- **–í—Å—Ç–∞–≤–ª—è—î** –ø—É–Ω–∫—Ç—É–∞—Ü—ñ—é, —è–∫
–í–∏ –º–æ–∂–µ—Ç–µ –¥–æ—Å—è–≥—Ç–∏ —Å–≤–æ—ó—Ö —Ü—ñ–ª–µ–π —Ñ–æ—Ä–º–∞—Ç—É–≤–∞–Ω–Ω—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –±—É–¥—å-—è–∫–æ–≥–æ –º–µ—Ç–æ–¥—É, –∞–ª–µ —à–∞–±–ª–æ–Ω–Ω—ñ –ª—ñ—Ç–µ—Ä–∞–ª–∏ –≤—Ä–∞—Ö–æ–≤—É—é—Ç—å —É—Å—ñ –ø—Ä–æ–±—ñ–ª–∏ —Ç–∞ —Ä–æ–∑—Ä–∏–≤–∏ —Ä—è–¥–∫—ñ–≤.

‚úÖ –ö–æ–ª–∏ –≤–∞—Ä—Ç–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —à–∞–±–ª–æ–Ω–Ω–∏–π –ª—ñ—Ç–µ—Ä–∞–ª, –∞ –∫–æ–ª–∏ –∑–≤–∏—á–∞–π–Ω–∏–π —Ä—è–¥–æ–∫?

### –ë—É–ª–µ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è
